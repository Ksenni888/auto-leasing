# АВТОЛИЗИНГ
# Описание.
-----------------------------------------------------------------------------------------
Консольное приложение на Java Core, без использования сторонних библиотек, которое имитирует автолизинг.

## Основные классы:

●	Car - автомобиль

      ○	Поля: VIN, марка, модель, год выпуска, стоимость, статус (доступен/в лизинге)

●	Client - клиент

      ○	Поля: ID, ФИО, номер паспорта, телефон

●	LeasingContract - лизинговый договор

      ○	Поля: номер договора, автомобиль, клиент, срок (мес.), первоначальный взнос, процентная ставка, статус (активен/закрыт), график платежей

●	Payment - платёж по договору

      ○	Поля: номер платежа, сумма, статус (оплачен/не оплачен)

## Функции приложения: 

Реализовать следующие операции через консольное меню:

●	Управление автомобилями:

      ○	Добавить новый автомобиль

      ○	Просмотреть список всех автомобилей

      ○	Найти автомобиль по критериям (марка, модель, год)

●	Управление клиентами:

      ○	Добавить нового клиента

      ○	Просмотреть список клиентов

      ○	Найти клиента по ФИО или паспорту

●	Управление договорами:

      ○	Создать новый лизинговый договор

      ○	Просмотреть активные договоры

    ■	Посмотреть любой договор по индексу или номеру договора

      ●	Выдает полную информацию об автомобиле, кроме доступности

      ●	Выдает полную информацию о клиенте

      ●	Выдает информацию лизингу (сумма финансирования, стоимость автомобиля, первоначальный взнос, срок в месяцах)

      ●	Выдает график платежей

      ●	Выдает информацию о проведенных платежах

      ○	Просмотреть историю договоров по клиенту

●	Управление платежами:

      ○	Зарегистрировать платеж по договору

      ○	Просмотреть историю платежей по договору

## Логика приложения.

●	При выборе каждого из меню загружает файлы (например выборе в меню работа с клиентами, загружает файл со списком всех клинтов и параметров)

●	При добавлении нового автомобиля или клиента происходит запись всей информации в уже имеющиеся файл

●	Создание договора:

     ○	Выбираем клиента(по id, ФИО или телефону)

     ○	Выбираем доступный автомобиль(по VIN)

     ○	Проверяем доступность автомобиля

     ○	Вводим параметры договора (срок, первоначальный взнос)

     ○	Система рассчитывает график платежей:

       ■	Сумма кредита = стоимость авто - первоначальный взнос

       ■	Ежемесячный платеж = аннуитетный платеж (рассчитывается по формуле)*

       ■	Создаются записи платежей на каждый месяц

     ○	Система создаёт номер договора

     ○	Автомобиль помечается как "в лизинге" 

     ○	Договор сохраняется

     ○	После внесения последнего платежа статус договора меняется на закрыт

●	Регистрация платежа:

     ○	Выбираем договор(по номеру)

     ○	Вводим сумму платежа

     ○	Система проверяет:

      ■	Соответствует ли сумма требуемому платежу

     ○	Обновляется статус платежа
